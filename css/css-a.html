关于我 (opens new window)

  • css-a

我的老家在农村，那是一个宁静，温馨的小村庄。在我的老家有奶奶，姑姑和叔叔，还有可爱的小弟弟。今天我陪着妈妈一起回老家看爸爸，也顺便体验一下丰富多彩的乡村生活。

伴着明媚的阳光，我和妈妈来到爸爸的绿色种植基地。远远望去，一排排蔬菜大棚在甬道两侧整整齐齐的排列着，洁白的塑料薄膜在阳光下散发着点点金光。走进棚里，一股浓重的混合着泥土清香的气息扑鼻而来。翠绿的西红柿映入我的眼帘。墨绿的枝叶，娇嫩的黄色小花，健康粗壮的筋脉，透出一股生机。还有那枝头上笑红了脸的西红柿果实，看着就有一种咬上一大口的冲动。我摩拳擦掌，撸起袖子，准备我的采摘。走进西红柿丛中，低头认真找寻着最大最红的果实，这些西红柿个个又大又红，都好像大将军一样使劲儿把头高高昂起，简直让我挑花了眼。好不容易找到了一个，兴奋地把它摘下来。西红柿捧在手里，感觉凉凉的，硬硬的，一股特有的芳香萦绕着我。红色伴着一丝丝的浅绿，妈妈说这是西红柿还没有熟透，等熟彻底了，就是紫红紫红的颜色了。一会儿工夫我就摘了十多个，我挑了一个最大的西红柿，迫不及待地冲洗干净，狠狠得咬了一口，酸酸的味道伴着甘甜在嘴巴里面迅速蔓延开来，唇齿之间顿时芳香四溢，真的太好吃了。吃着西红柿，看着辛勤劳动的人们，我满心欢喜和激动。我品尝的就是老爸一直努力的辛勤成果。真为老爸的伟大而自豪。

这个时候老爸过来，手里拿着一颗绿色的东西，问我：知道这是什么吗？我迷糊地摇头说着不知道。然后老爸告诉我说：“是一种可以吃的野菜。我带你去挖，中午咱们吃这个。”

我和妈妈在爸爸的带领下拿着工具出发了。可是走啊走啊，到处是光秃秃的土地，连个绿色的小草都没有，哪里来的野菜啊。爸爸笑着和我说：“不要急，要有耐心。野菜长在树林里，而且很难发现的，这要看你的火眼金睛喽！”我们走进前面的树林里面，我迫不及待地冲了进去，开始仔细地寻找野菜。找呀，找呀，过了好长时间，还是没找到一棵野菜。当我灰心的时候，妈妈大叫了一声：妞子，这里有一棵荠菜，你快来拔吧！”于是，我马上跑了过去，把那棵荠菜连根拔起。我仔细观察了荠菜样子，荠菜的叶子是绿油油，就像一根一根的锯齿，然后我又开始照着这个样子仔细地去寻找了，找着，找着，突然，我惊喜地大叫起来：“妈妈，我找到了好多荠菜呢，赶快过来拔吧！”妈妈兴冲冲地跑了过来，和我一起拔了起来。有的荠菜大摇大摆地长在光秃秃的土层上，我一下子就把它拔了出来；有的荠菜在玩捉迷藏的游戏，躲在小草中间，但是依旧过我的火眼金睛，都被我找到了。很快，我们就拔满了一大塑料袋，看着亲手拔的绿绿的荠菜，我的心里乐开了花。

时间过的真快，午饭时间到了。我们伴随着小鸟的动听的叫声回家了。这次回老家体验生活，虽然辛苦，但是我却非常真实的感觉到了劳动的快乐。我爱这丰富多彩的乡村生活！
